🧮 Manufacturability Score Calculation

The manufacturability score measures how well each clone performs compared to the average across all parameters:

📊 Step-by-Step Process:

  1. Calculate Column Averages: For each parameter (VCD D7, VCD D12, etc.), calculate the average value across all clones
  2. Compare Each Clone: For each clone, compare its value to the column average:
    • If clone value > average: +1 point
    • If clone value ≤ average: +0 points
  3. Sum the Points: Add up all the points for each clone across all parameters
  4. Final Score: The total points = Manufacturability Score

💡 Example Calculation:

Scenario: Clone A with 3 parameters analyzed

Parameter Clone A Value Average Points
VCD D12 24.2 20.5 1 (24.2 > 20.5)
Titer D12 1802 1850 0 (1802 ≤ 1850)
QP 47.2 35.0 1 (47.2 > 35.0)
Total Score 2 out of 3

🎯 Key Points:

  • Score Range: 0 to N (where N = number of analyzed parameters)
  • Higher is Better: Higher scores indicate better manufacturability
  • Relative Ranking: Scores are relative to your dataset's performance
  • Missing Data: Parameters with no data are excluded from scoring
  • Fair Comparison: All clones are scored against the same averages
